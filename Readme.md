# Spring Framework Project
>Do not use main branch for running the code.

**Use the following branches to test various features of Spring Framework**

> java-config
>- It demonstrates configuring and using Spring Framework using Java.

> xml-config
>- It demonstrates configuring and using Spring Framework using XML configuration.

> java-config-bean-scope
>- It demonstrates bean scopes: singleton and prototype.

> java-config-constructor-inject
>- It demonstrates injecting dependency using constructor injection.

> java-config-setter-inject
>- It demonstrates injecting dependency using setter injection.

> java-config-qualifier
>- It demonstrates resolving injected dependency using qualifier.

> java-config-lazy
>- It demonstrates creating beans using lazy initialization.

> java-config-lifecycle
>- It demonstrates using init and destroy lifecycle methods.

> java-config-bean-method
>- It demonstrates creating bean using @Bean method.
